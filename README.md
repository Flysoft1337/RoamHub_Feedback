## English

### Welcome to RoamHub Feedback Center!

This is the official feedback repository for the RoamHub community. We welcome all users to submit issues, suggestions, and feedback to help us improve RoamHub.

### About RoamHub

RoamHub is a community client based on Jetpack Compose, utilizing modern Android development technology to provide users with a smooth and elegant community interaction experience.

### How to Submit Feedback

You can submit feedback in the following ways:

#### ğŸ› Bug Reports
If you encounter issues while using RoamHub, please use the [Bug Report template](../../issues/new?template=bug_report.md) to submit a detailed problem description.

#### ğŸ’¡ Feature Requests
If you have ideas or suggestions for new features, please use the [Feature Request template](../../issues/new?template=feature_request.md) to tell us.

#### ğŸ“ General Feedback
For other types of feedback or questions, please use the [General Feedback template](../../issues/new?template=general_feedback.md).

### Feedback Guidelines

To help us better handle your feedback, please:

1. **Search existing issues** - Before submitting a new issue, please search for similar existing issues
2. **Use templates** - Choose the appropriate issue template and fill in all required information
3. **Clear description** - Provide clear, detailed descriptions, including reproduction steps (if applicable)
4. **Attach screenshots** - If necessary, attach screenshots or videos to illustrate the issue
5. **Stay polite** - Respect others and maintain a friendly communication atmosphere

### Community Guidelines

- Respect all community members
- Do not post malicious, offensive, or inappropriate content
- Avoid submitting duplicate issues
- Follow GitHub Community Guidelines
