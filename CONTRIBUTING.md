## English

Thank you for considering contributing to RoamHub!

### How to Contribute

#### Reporting Issues
1. Use GitHub Issues search to check if a similar issue already exists
2. If not found, create a new Issue
3. Use the appropriate template (Bug Report, Feature Request, or General Feedback)
4. Provide as much detailed information as possible

#### Submitting Feedback
- Clearly describe your suggestion
- Explain why this change would be useful
- Provide usage examples if possible

### Code of Conduct

#### Our Pledge
In the interest of fostering an open and welcoming environment, we pledge to make participation in our project and community a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

#### Our Standards
Examples of behavior that contributes to creating a positive environment include:
- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the community
- Showing empathy towards other community members

Examples of unacceptable behavior include:
- The use of sexualized language or imagery and unwelcome sexual attention or advances
- Trolling, insulting/derogatory comments, and personal or political attacks
- Public or private harassment
- Publishing others' private information, such as physical or electronic addresses, without explicit permission
- Other conduct which could reasonably be considered inappropriate in a professional setting

### Issue Labels

- `bug` - Confirmed defects or errors
- `enhancement` - New feature or improvement requests
- `feedback` - General feedback or suggestions
- `documentation` - Documentation-related improvements
- `question` - Requires additional information or discussion
- `duplicate` - Duplicate issues
- `wontfix` - Issues that won't be fixed
- `help wanted` - Issues that need community help

### Response Time

We strive to respond to new Issues within **48 hours**. However, please note that RoamHub is a community-maintained project, and response times may vary.

### Contact

If you have any questions, please:
- Ask in GitHub Issues
- Join our community discussions
